# 📈 KOSDAQ Low-PSR Quant Screener & Backtester
> 한국 주식 시장(코스닥)에서 **매출 대비 저평가(낮은 PSR)** 상태인 우량 소형주들을 발굴하고, 과거 성과를 정밀하게 시뮬레이션할 수 있는 Streamlit 기반 퀀트 투자 플랫폼입니다.

---

## 📊 Backtest Performance (2024.05 ~ 2026.06)

* **전략 조건**: 주가매출비율(PSR) < 0.8, 부채비율 < 100%, 최근 3분기 연속 영업이익 흑자(>0) 유지
* **운용 규칙**: 매 분기별 리밸런싱(공시 기간을 고려한 Look-ahead Bias 방지 반영), PSR 최저 상위 10개 종목 균등 분산 투자, 매매 수수료 및 거래세 0.2% 차감

| 성과 지표 (Metric) | 퀀트 포트폴리오 (Quant) | 코스닥 지수 (Benchmark) | 초과 성과 (Alpha) |
| :--- | :---: | :---: | :---: |
| **누적 수익률 (Total Return)** | **+20.5%** | **+6.1%** | **+14.4%p** |
| **연평균 복리수익률 (CAGR)** | **+9.5%** | **+2.9%** | **+6.6%p** |
| **최대 낙폭 (MDD)** | **-15.9%** | **-28.0%** | **+12.1%p (리스크 방어)** |
| **샤프 지수 (Sharpe Ratio)** | **0.56** | - | - |

> **💡 전략 검증 시사점**: 코스닥 시장이 급락하며 벤치마크 지수가 -28.0%의 최대 낙폭을 보였을 때, 본 전략은 재무 안전성 필터와 분산 효과를 통해 낙폭을 **-15.9%로 반감**시켰습니다. 이는 코스피 비중을 줄이고 안전한 알파(초과수익)를 얻고자 하는 바스켓 투자 전략에 최적화된 결과입니다.

## ✨ 주요 기능 (Key Features)

1. **📊 실시간 종목 스크리너 (Screener)**
   - 사용자가 임의로 지정하는 PSR 상한선, 부채비율 상한선, 연속 흑자 분기 수 필터를 적용하여 실시간으로 조건을 충족하는 코스닥 우량 저평가주 목록 제공.
   - 선정된 종목 리스트 테이블 제공 및 즉시 다운로드(CSV 내보내기) 기능 지원.
   - 포트폴리오 바스켓 후보군을 시각화 카드로 직관적으로 추천.

2. **📈 성과 분석 백테스터 (Backtester)**
   - 시작일, 종료일, 포트폴리오 편입 종목 수, 리밸런싱 주기(월별, 분기별, 반기별, 연도별), 투자 원금을 설정하여 백테스트 시뮬레이션 가동.
   - 코스닥 지수 대비 포트폴리오의 실시간 누적 수익률 추이를 **Plotly 대화형 차트**로 정교하게 시각화.
   - 시뮬레이션 기간 동안 발생한 모든 매수/매도 세부 거래 기록(로그)을 제공하여 투명성 확보.

3. **⚙️ 비차단 금융 데이터 캐시 엔진 (Data Manager)**
   - FinanceDataReader 및 네이버 금융을 활용해 전 종목 종가 및 4분기/연간 재무 데이터를 자동 수집 및 로컬 파일(`data/`)에 압축 캐싱.
   - 비동기 스레드 풀(Thread Pool, 40 workers) 관리 개선 및 **비차단 셧다운 방식(`shutdown(wait=False)`)**을 도입하여 네트워크 대기 지연 없이 약 1분 20초 이내에 데이터 수집 강제 완료 및 빌드 보장.
   - Streamlit 프로그레스 바(Progress Bar)를 통해 데이터 수집 진척률을 실시간 시각화.

## 🛠️ 설치 및 실행 방법 (Installation & Run)

### 1. Repository Clone
```bash
git clone https://github.com/Hakhyun-Kim/Quant.git
cd Quant
```

### 2. 패키지 설치 (Install Dependencies)
```bash
pip install -r requirements.txt
```

### 3. 대시보드 애플리케이션 실행 (Run App)
```bash
python -m streamlit run app.py
```
실행이 완료되면 자동으로 웹 브라우저 창(`http://localhost:8501`)이 열리며 분석을 시작할 수 있습니다.
